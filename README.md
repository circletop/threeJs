# 简介
本仓库是学习`ThreeJs`的demo练手库，欢迎👏👏👏各位在学习的小伙伴一起探讨、研究。
## 目录介绍

```bash
.
├── dist # 文件打包目录
├── package-lock.json # 依赖版本锁定
├── package.json # 项目依赖目录
└── src 
    ├── assets  # 静态文件目录
    │   ├── css
    │   └── imgs
    ├── index.html # 项目入口
    └── main # demo 目录
        ├── main.js
        ├── main01.js
        ├─ ...

```
## 依赖说明

```bash
npm i gsap --save  # 动画库
npm i dat.gui --save  # gui 开发调试
```
